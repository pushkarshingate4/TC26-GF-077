# TC26-GF-077

DetachX Car – Modular Smart STEM Car

Team ID: TC26-GF-077
Domain: Electronics

Project Overview

DetachX Car is a modular smart STEM prototype designed to demonstrate concepts of electronics, embedded systems, sensors, and automation through a detachable learning platform.

Unlike traditional robotic kits, the system allows motors, sensors, and control modules to be magnetically attached or detached, enabling students to explore real-world engineering concepts interactively.

System Architecture

The DetachX Car uses a layered modular architecture:

1️⃣ Control Layer – ESP32 microcontroller
2️⃣ Sensing Layer – Ultrasonic, IR, and LDR sensors
3️⃣ Actuation Layer – Motor units
4️⃣ Power Layer – Rechargeable battery

Each module connects through magnetic connectors with copper contact interfaces, allowing easy attachment and electrical connectivity.
